# 🛡 Network Monitor

## 🇹🇷 Türkçe

📌 Proje Açıklaması:
Bu proje, yerel ağda cihazları tespit eden, IP-MAC eşleşmesini takip eden ve ARP tabanlı olası saldırıları algılayan basit bir ağ izleme sistemidir. Flask tabanlı bir web panel üzerinden canlı log, cihaz listesi ve uyarılar gösterir.

⚙️ Özellikler:
- Gerçek zamanlı ağ cihazı tespiti
- IP ve MAC adres takibi
- ARP değişim tespiti
- Canlı log akışı
- Web dashboard (Flask)

🚀 Çalıştırma:
python main.py
python app.py

Tarayıcı:
http://127.0.0.1:5000

🧠 Amaç:
SOC mantığını simüle etmek ve ağ izleme sistemlerini öğrenmek.

---

## 🇬🇧 English

📌 Project Description:
This project is a simple network monitoring system that detects devices in a local network, tracks IP-MAC relationships and detects possible ARP-based attacks. It provides a real-time Flask dashboard.

⚙️ Features:
- Real-time device detection
- IP & MAC tracking
- ARP spoof detection
- Live log stream
- Flask web dashboard

🚀 How to Run:
python main.py
python app.py

Browser:
http://127.0.0.1:5000

🧠 Purpose:
To simulate a basic SOC network monitoring system.
